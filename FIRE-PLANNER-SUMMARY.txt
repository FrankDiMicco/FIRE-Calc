FIRE PLANNER - PROJECT SUMMARY
================================

VISION
------
Comprehensive, modular FIRE planning tool that integrates budget, taxes, ACA subsidies, 
retirement simulations, and portfolio strategy - all working standalone OR connected together.


CORE MODULES
------------
1. Budget Builder (foundation - feeds other tools)
2. Years to FI Calculator
3. Retirement Simulator (historical + Monte Carlo)
4. Bucket Strategy Builder
5. Tax Calculator
6. ACA Subsidy Calculator
7. Roth Conversion Optimizer
8. Withdrawal Strategy Comparison
9. Integrated Dashboard

PATH FORWARD
------------
Phase 1: Build web app with card dashboard
  - Implement card-based navigation
  - Create shared state manager
  - Build data import/export system

Phase 2: Implement core modules one at a time
  - Start with Budget Builder (foundation)
  - Add Years to FI Calculator
  - Add Retirement Simulator with Shiller data
  - Continue with remaining modules

Phase 3: Add integration layer
  - Module-to-module data sharing
  - Visual connection indicators
  - Dashboard that synthesizes all data

Phase 4: Polish and test
  - Mobile optimization
  - PWA features (offline, installable)
  - User testing and refinement

Phase 5: (Future) iOS app
  - Wrap with Capacitor
  - Build in Xcode
  - Submit to App Store


TECHNICAL BEST PRACTICES FOR FUTURE iOS
----------------------------------------
- Use relative units (rem, %, vh) not fixed pixels
- Touch targets minimum 44px (iOS standard)
- Avoid hover-dependent interactions
- Keep file paths relative
- Structure code in clean modules
- Use custom modals instead of alert()/confirm()
- Don't rely on browser-specific features
